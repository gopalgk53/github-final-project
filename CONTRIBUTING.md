# Contributing to Simple Interest Calculator

We welcome contributions from everyone! 🎉  
All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

1. **Fork the Repository**  
   Create your own copy of the project to work on.

2. **Create a Branch**  
   Use a descriptive branch name for your changes:
   ```bash
   git checkout -b bug_fix_typo
3. Make Your Changes
	◦ Fix bugs
	◦ Add new features
	◦ Improve documentation
	◦ Suggest enhancements
4. Test Your Changes
Ensure the script runs correctly and outputs expected results.
5. Submit a Pull Request
Push your branch and open a pull request with a clear description of your changes.
Contribution Guidelines
• Keep code clean and well-documented.
• Follow the existing style and structure of the project.
• Provide examples or test cases when adding new features.
• Be respectful and constructive in discussions.
Reporting Issues
If you find a bug or have an idea for improvement:
• Open an issue in the repository.
• Provide details such as steps to reproduce, expected behavior, and screenshots if applicable.
