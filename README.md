# github-final-project

## 📖 Description

The **Simple Interest Calculator** is a Bash script designed to quickly compute simple interest based on user-provided values for **Principal**, **Rate of Interest**, and **Time Period**.  

It applies the standard formula:

\[
SI = \frac{P \times R \times T}{100}
\]

where:
- **P** = Principal amount  
- **R** = Rate of interest (per annum)  
- **T** = Time period (in years)

### 🎯 Purpose
This project is designed to:
- Provide a lightweight and easy-to-use financial calculator.
- Help students, finance learners, and professionals understand and apply the concept of simple interest.
- Demonstrate how Bash scripting can be used for practical, real-world calculations.
